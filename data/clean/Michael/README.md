# Cleaned Data on (GHG) Greenhouse gas emissions in Canada's Provinces and Territories covering from 2013 to 2023.

* Source is from "Statistics Canada".



## GHG Emissions — Clean Data

* The cleaned GHG emissions dataset was prepared using Python (Pandas)
*
* The data was transformed into structured analytical tables focused on:
* Province-level GHG emission trends across Canada
* Nova Scotia sector-level emissions analysis

### The cleaning process included:
* Removing unnecessary metadata columns
* Filtering relevant provinces and sectors
* Standardizing column names and data types
* Removing duplicates and null values
* Simplifying sector labels for visualization

The goal for the cleaned datasets are for optimized trend analysis, KPI calculations, province comparisons, forecasting, and Power BI dashboard development.
