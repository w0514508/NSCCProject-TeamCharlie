
## Cleaned Dataset for Air Quality Analyis
##### This dataset was created from the raw air‑quality data downloaded from the Government of Canada Open Data Portal. The raw files were cleaned to remove null and invalid values and then appended into a single staging dataset for analysis.

##### The staging table includes observations for three pollutants and five monitoring stations, with records spanning 1995 to 2025, depending on data availability by pollutant and station.
