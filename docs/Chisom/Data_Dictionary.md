# Data Modeling & Preparation Process
Loaded and cleaned Nova Scotia air quality CSV data in Power BI using Power Query.
Defined the fact table grain as daily air quality measurements by station and pollutant.
Built a star schema by creating dimension tables for Date, Station, and Pollutant 
Enriched dimensions with descriptive attributes (e.g., pollutant names and units).
Established one to many relationships with fact table
Created and marked a custom Date table with a hierarchical structure for time based analysis.
