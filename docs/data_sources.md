# Data Sources

This document lists the data sources used in the NSCC Capstone Project.

## Monthly Historical Climate Data

Data Source: Monthly Historical Climate Data
Provider:
Environment and Climate Change Canada (ECCC)
Access Method:
Historical Climate Data – Monthly Climate Data (CSV download via ECCC portal)
Dataset Description:
Monthly consolidated climate observations derived from daily station records. The dataset includes monthly totals and averages such as total precipitation, mean temperatures, and observed extremes.
Geographic Scope:
Canada — focus on Nova Scotia


### Weather Stations Used

The precipitation analysis is based on data from the following Environment and Climate Change Canada (ECCC) climate stations in Nova Scotia:

- Shearwater (NS) — ECCC Climate Station (Climate ID: 8205090)
- Yarmouth (NS) — ECCC Climate Station (Climate ID: 8206500)
- Sydney (NS) — ECCC Climate Station (Climate ID: 8205700)

Stations were selected based on data availability, historical coverage, and geographic representativeness. Climate IDs are used as unique identifiers to ensure consistency across data sources and time periods.

Time Coverage: varies by station: Shearwater(1944-2007), Yarmouth(1940-2012), Sydney(1941-2014)
Granularity:
Monthly
Key Variable Used in This Project:

Total Precipitation (mm)

Intended Use in the Project:
This dataset is used as the primary source for long‑term precipitation trend and seasonal analysis due to its long historical coverage and consolidated monthly format.
Notes / Limitations:

Monthly data is not available for recent years (post‑2012) for many stations.
Data values are already aggregated from daily observations using standard ECCC methodologies.
